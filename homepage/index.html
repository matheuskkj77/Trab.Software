Death Cow

Desenvolvemos este programa com o objetivo de proporcionar uma experiência mais eficiente e organizada para estabelecimentos gastronômicos. O projeto foi originalmente concebido como trabalho escolar, mas foi posteriormente expandido para possibilitar a sua utilização diária pelos restaurantes. Com a implementação deste programa, buscamos otimizar os processos operacionais dos restaurantes, como o gerenciamento de pedidos, controle de garçons e administração de mesas. Ademais, o nosso sistema foi projetado de forma a ser intuitivo e de fácil utilização, com uma interface amigável faceis de se navegar.
